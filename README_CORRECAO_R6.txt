V20.8.23 R6 - Correção O.S. WhatsApp/PDF

Correções aplicadas:
- Removido botão BAIXAR PDF da tela de impressão da O.S.
- Mantidos apenas botões de envio/abertura via WhatsApp:
  - PDF WhatsApp Prestador
  - PDF WhatsApp Responsável Loja
- Removidos botões repetidos de WhatsApp simples na O.S.
- O sistema tenta compartilhar o PDF como arquivo usando o recurso nativo do navegador.
- Quando o navegador/WhatsApp Web não permite anexar arquivo automaticamente, abre apenas a conversa do WhatsApp, sem enviar link do sistema.
- Botões COM VALOR e SEM VALOR continuam funcionando e são respeitados ao gerar o PDF.
